Português
Specific_genes.py foi feito para receber o arquivo csv de dna (por enquanto apenas para o schema do genera) 
e bater as informações (alelos dos genes) do arquivo com os alelos de artigos publicados no snpedia, retornando uma possível "patologia".

English
Specific_genes.py was made to receive the dna csv file (for now only for genera schema)
and match the information (gene alleles) from the file with the alleles from articles published on snpedia, returning a possible "pathology".
